# 快速构建 MineContext 安装程序

## 一键构建

```bash
# 1. 首先构建前端应用
cd frontend
npm run build:win

# 2. 返回项目根目录，构建安装程序
cd ..
build-installer.bat
```

## 详细步骤

### 步骤 1: 构建前端应用 (已构建可跳过)

如果你的 `build-output\MineContext-Windows\MineContext.exe` 已经存在，可以跳过此步骤。

```bash
cd frontend
npm install
npm run build:win
```

### 步骤 2: 构建安装程序

双击运行 `build-installer.bat`，或在命令行中执行：

```bash
build-installer.bat
```

### 步骤 3: 获取安装程序

构建完成后，安装程序位于：

```
build-output\MineContext-Setup-0.1.2-beta.exe
```

## 安装程序特性

### ✓ 可选功能

- **桌面快捷方式** - 可选创建
- **开机自动启动** - 可选启用 (重要功能!)
- **快速启动栏快捷方式** - 可选创建 (Windows 旧版本)

### ✓ 自动功能

- **开始菜单项** - 自动创建
- **程序卸载** - 自动创建卸载程序
- **配置文件夹** - 自动创建

## 用户安装步骤

1. 运行 `MineContext-Setup-0.1.2-beta.exe`
2. 选择安装位置 (默认: `C:\Program Files\MineContext`)
3. **重要**: 勾选"开机自动启动"选项 (如果希望开机自动启动)
4. 选择是否创建桌面快捷方式
5. 点击"安装"
6. 安装完成后，程序会自动启动 (如果选择了相应选项)

## 开机启动说明

### 启用开机启动

在安装时勾选"开机自动启动"选项，或者在安装后：

1. 按 `Win + R` 打开运行对话框
2. 输入 `shell:startup` 并回车
3. 将 `MineContext.exe` 的快捷方式拖入此文件夹

### 禁用开机启动

1. 按 `Ctrl + Shift + Esc` 打开任务管理器
2. 点击"启动"标签
3. 找到"MineContext"并点击"禁用"

或者：

1. 按 `Win + R` 打开运行对话框
2. 输入 `regedit` 并回车
3. 导航到: `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
4. 删除 `MineContext` 项

### 卸载时自动清理

卸载程序时，会自动：
- ✓ 删除所有程序文件
- ✓ 删除开机启动项
- ✓ 删除注册表项
- ✓ 删除快捷方式

## 故障排除

### 安装程序无法运行

**原因**: 需要管理员权限

**解决**: 右键点击安装程序，选择"以管理员身份运行"

### 构建失败

**错误**: "找不到 Inno Setup 6"

**解决**: 确认 Inno Setup 6 安装在 `E:\Program Files (x86)\Inno Setup 6`

**错误**: "未找到已构建的应用程序"

**解决**: 先运行 `cd frontend && npm run build:win`

### 开机启动不工作

**检查**:
1. 安装时是否勾选了"开机自动启动"选项
2. Windows 防火墙或杀毒软件是否拦截
3. Windows 启动项是否被组策略禁用

**验证**:
1. 按 `Win + R`
2. 输入 `msconfig` 并回车
3. 在"启动"标签中查看是否有 MineContext

## 技术细节

### 安装路径

```
C:\Program Files\MineContext\
├── MineContext.exe          # 主程序
├── *.dll                     # 依赖库
├── locales\                  # 语言包
├── resources\                # 应用资源
│   ├── app.asar             # 打包的应用文件
│   └── app.asar.unpacked\   # 解压的资源
└── config\                   # 配置文件目录
```

### 开机启动注册表

**位置**: `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
**键名**: `MineContext`
**值**: `"C:\Program Files\MineContext\MineContext.exe"`
